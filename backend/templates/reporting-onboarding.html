1. Your Daily Code (Every Weekday at 9 AM)

Every Monday through Friday at 9:00 AM Eastern, you'll receive an email with a unique 6-character code (for example: X7K2M9). This is your clock-in code for that day. Each code can only be used on the day it's sent and expires at midnight.

2. Clocking In

When you're ready to start work, go to:

{{portalUrl}}

Enter your 6-character code from the email and click Clock In. That's it — no username, no password needed. You'll see a confirmation with the time you clocked in.

Tip: If you accidentally close the browser or switch devices, just go back to the same URL and enter the same code again. It will bring you right back to where you left off (as long as you haven't clocked out yet).

3. Logging Your Work (3 Time Slots)

Your workday is split into 3 time slots, each 2 hours long:

Slot 1 Available immediately when you clock in
Slot 2 Unlocks 2 hours after you clock in
Slot 3 Unlocks 4 hours after you clock in

In each slot, log what you worked on as bullet points. For each task you can also set a time range (from/to) so management can see when you worked on what.

How to add tasks:

  • Click the Add another task button to add a new bullet point
  • Type your task description (e.g., "Called 5 leads from the new list")
  • Optionally set the start and end time for that task
  • Press Enter to quickly add another bullet
  • Press Backspace on an empty bullet to remove it

4. Saving & Submitting

Auto-save: Your work is saved automatically every 30 seconds, so you won't lose anything if your browser closes.

Submitting a slot: When you're done with a slot, click the Submit button on that slot. Try to submit each slot within its 2-hour window — if you submit after the window, it'll be marked as late (management can see this).

5. Reminders

If you haven't submitted a slot by the time the next one unlocks, you'll get a reminder email nudging you to submit. After Slot 3's deadline, you'll get a final reminder to wrap up and clock out.

6. Clocking Out

When you're done for the day, click the Clock Out button at the bottom of the page. This will:

  1. Record your clock-out time
  2. Compile all your tasks into a daily report
  3. Generate an AI summary of your work
  4. Send the summary to management

Important: Once you clock out, your code is used up for the day. You won't be able to log back in or make changes, so make sure all your tasks are entered before clocking out.
